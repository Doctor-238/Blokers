========================================
BLOKERS JFRAME SOCKET GAME - 프로젝트 상태
========================================

프로젝트: 블로커즈 (Blokers)
설명: JFrame 기반 소켓 멀티플레이어 보드 게임
상태: ✅ 완료 및 검증됨

========================================
구현 완료 사항
========================================

✅ 서버 구현 (BlokusServer.java)
   - 포트 12345에서 실행되는 소켓 서버
   - 멀티스레드 클라이언트 처리
   - 게임방 생성 및 관리
   - 플레이어 인증 시스템

✅ 클라이언트 구현 (BlokusClient.java)
   - JFrame GUI (800x830)
   - 4개 화면: 로그인, 로비, 방, 게임
   - CardLayout 기반 화면 전환
   - 실시간 네트워크 통신

✅ 게임 로직 (GameRoom.java)
   - 20x20 보드 관리
   - 턴 기반 게임플레이
   - 유효성 검증 (블로커스 규칙)
   - 타이머 시스템 (5분 + 10초/턴)
   - 점수 계산 및 승자 결정

✅ UI 컴포넌트 (GameScreen.java)
   - 게임 보드 렌더링 (500x500)
   - 블록 인벤토리 (스크롤 가능)
   - 고스트 조각 미리보기
   - 마우스/키보드 상호작용
   - 채팅 시스템

✅ 게임 조각 (BlokusPiece.java)
   - 21가지 고유 조각 정의
   - 회전 기능 (90도)
   - 좌표 계산

✅ 네트워크 프로토콜 (Protocol.java)
   - C2S: LOGIN, CREATE_ROOM, JOIN_ROOM, PLACE, PASS_TURN, CHAT 등
   - S2C: GAME_START, GAME_STATE, HAND_UPDATE, TIME_UPDATE 등

========================================
실행 방법
========================================

1. 서버 시작
   ./start-server.sh
   또는
   ./gradlew runServer

2. 클라이언트 시작 (각각 새 터미널)
   ./start-client.sh
   또는
   ./gradlew runClient

3. 게임 플레이
   - 이름 입력하여 로그인
   - 방 생성 또는 참가
   - 2명 또는 4명이 모이면 시작
   - 조각 선택 후 보드에 배치
   - R 키로 회전

========================================
주요 기능
========================================

✅ 멀티플레이어
   - 2인 (1v1): 각 플레이어 2색상
   - 4인: 각 플레이어 1색상

✅ 게임 규칙
   - 첫 이동: 시작 코너 포함
   - 이후: 대각선 접촉, 변 접촉 금지
   - 승리: 남은 조각 최소

✅ 타이머 시스템
   - 색상당 5분 초기 시간
   - 턴마다 10초 추가
   - 시간 초과 시 비활성화

✅ 실시간 기능
   - 채팅
   - 보드 상태 동기화
   - 타이머 업데이트
